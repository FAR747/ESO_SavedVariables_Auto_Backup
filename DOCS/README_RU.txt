Спасибо за загрузку ESO SavedVariables Auto Backup!

Как установить:
  1. Распакуйте архив в удобное место. (Я настоятельно ** рекомендую ** просто распоковать в папку `/Documents/Elder Scrolls Online/`)
  2. Запустите `ESO SavedVariables Auto backup.exe`.
  3. В открывшемся окне *нажмите "Далее"*
  4. Выберите каталог Elder Scrolls Online в каталоге документов. Сама программа должна найти этот каталог и показать папки ESO *(live, pts и так далее)* *Нажмите Next*
  5. Здесь нужно выбрать место, куда будут сохраняться резервные копии. Советую оставить как есть.
Ниже выберите папки ESO, из которых будет выполнено резервное копирование настроек. (По умолчанию выбраны все папки, но обратите внимание, что вы не можете снять отметку с папки `live`) *Нажмите Далее*
  6. Все настройки завершены. Теперь *нажмите Готово*

Как обновить программу:
Загрузите архив с https://github.com/FAR747/ESO_SavedVariables_Auto_Backup/releases и распакуйте его туда, где установлена программа.
(Если вы получили уведомление о новой версии и нажали кнопку «Открыть страницу обновления», нужная папка откроется автоматически.)

Как настроить автоматическое резервное копирование:
 1. Нажмите «Файл» вверху и выберите «Настройки».
 2. В открывшемся окне вы найдете настройки автоматического резервного копирования:
`Automatic backup at program start` - если этот параметр включен, программа будет создавать резервные копии при каждом запуске. Хорошо сочетается с параметром `Run the program at Windows startup`.
`Automatic backup when closing ESO` - если этот параметр включен, программа будет создавать резервные копии через некоторое время после закрытия игры.
 3. Вы также можете настроить автоматическое удаление старых резервных копий, включив параметр `Remove old backups automatically` и указав ниже количество дней хранения резервной копии.
 Резервные копии будут автоматически удалены при запуске программы.
 Вы также можете вручную удалять бэкапы нажав правым кликом по нужному бекапу -> delete
